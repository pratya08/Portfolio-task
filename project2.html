In an automatic plant watering system or we can say it smart irrigation system. we are using an Arduino microcontroller to control and sense the thing in the projects such as sense or measure the moisture in the soil we are using soil moisture sensor. the soil moisture sensor is a sensor which varies the value when it contacts the moisture. basically it is a resistor that works on moisture condition. when the moisture is more then the value of resistance will decrease and when the moisture is less then the resistance value is more.
